# 智简笔记本 (WitNote) 安装说明

## 📦 安装步骤

1. **拖拽安装**
   将左侧的「智简笔记本」应用图标拖拽到右侧的 Applications 文件夹即可完成安装。

2. **首次启动**
   - 打开「启动台」或「访达 → 应用程序」
   - 找到「智简笔记本」并双击打开
   - macOS 可能会提示"无法验证开发者"，请按以下步骤处理：
     1. 打开「系统设置 → 隐私与安全性」
     2. 在底部找到相关提示，点击「仍要打开」
     3. 再次打开应用即可

3. **开始使用**
   - 首次启动后，点击「链接本地文件夹」选择一个文件夹存放笔记
   - 应用不会上传任何数据到云端，所有内容都保存在本地

---

## 🔧 系统要求

- macOS 11.0 (Big Sur) 或更高版本
- Apple Silicon (M1/M2/M3) 或 Intel 处理器
- 建议至少 4GB 可用内存（AI 功能需要更多内存）

---

## ❓ 常见问题

**Q: 应用打开后显示空白？**
A: 请等待几秒钟，首次启动需要加载资源。

**Q: AI 助手不工作？**
A: 内置 AI 需要首次下载模型，请保持网络连接。也可以安装 Ollama 使用更强大的本地模型。

**Q: 如何卸载？**
A: 将应用从「应用程序」文件夹拖到废纸篓即可。笔记数据不会被删除。
